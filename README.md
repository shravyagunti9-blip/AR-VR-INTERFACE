The Home Screen is the first page of the AR Furniture Application. It allows users to select furniture items such as Sofa, Table, Plant, and Lamp. Each item is displayed with an image and name for easy identification.

When the user selects a furniture item, the app moves to the AR Camera Screen, where the selected object can be placed in the real environment.

The Home Screen is simple, user-friendly, and designed for easy navigation.
